Image lazy loader plugin for Prototype.

Plugin enabled on container

Images below the visible area (the ones lower than container bottom) are not loaded. When scrolling down they are loaded when needed. Shift-reload to test again. Compare this to page where plugin is disabled or same page with fadein effect. 

lazyload.init({
     placeholder: "img/grey.gif",
     container: $('container')
});
